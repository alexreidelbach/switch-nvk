## 🇧🇷 Português

Este projeto é software livre e **deve permanecer assim**.

Qualquer pessoa que fizer um *fork*, modificar, integrar, ou redistribuir este
projeto — seja em código-fonte, seja em forma binária/compilada — é **obrigada a**:

1. **Manter tudo completamente open source.** Todo o código-fonte
   correspondente — incluindo quaisquer modificações, adições e o código
   necessário para compilar e rodar — deve ser disponibilizado publicamente e de
   forma gratuita.
2. **Propagar estes mesmos termos** (copyleft). Qualquer *fork* ou trabalho
   derivado deve ser licenciado sob estes mesmos termos, de modo que a obrigação
   de permanecer aberto acompanhe todos os derivados.
3. **Não** distribuir versões closed-source, ofuscadas, somente-binário, ou com
   o código-fonte oculto, restrito ou pago.

Estes termos se aplicam a todas as contribuições originais feitas neste
repositório. Partes derivadas de projetos *upstream* permanecem sob suas
licenças originais; esta política **se soma a** — e não remove — os direitos
concedidos por essas licenças.

## 🇬🇧 English

This project is free/open-source software and **must stay that way**.

Anyone who forks, modifies, integrates, or redistributes this project — whether
as source code or in binary/compiled form — **must**:

1. **Keep everything fully open source.** All corresponding source code —
   including any modifications, additions, and the code required to build and
   run it — must be made publicly available, free of charge.
2. **Carry these same terms forward** (copyleft). Any fork or derivative work
   must be licensed under these same terms, so that the obligation to stay open
   travels to every derivative.
3. **Not** distribute closed-source, obfuscated, binary-only, or
   hidden/restricted/paywalled-source versions.

These terms apply to all original contributions made in this repository.
Portions derived from upstream projects remain under their original licenses;
this policy **adds to** — and does not remove — the rights granted by those
licenses.
